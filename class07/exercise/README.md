# In Class Exercise

Build a small progressively enhanced page.

Create a page with: (use minimal CSS styling afterwards)

- heading
- short description
- button
- status paragraph
- content section with placeholder text

When the button is clicked:

- update the status to loading
- fetch one post from JSONPlaceholder
- replace the placeholder text with:
  - title
  - body
- show a success or error message

JSONPlaceholder link:
`https://jsonplaceholder.typicode.com/posts/1`
