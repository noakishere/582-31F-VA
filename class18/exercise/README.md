## in-class exercise

Create a page that displays a list of football players using a template inside.

### Requirements:

- one `<template>`
- one container
- one array of player objects
- one helper function that creates and fills each player component

#### Data example:

```
const players = [
  { name: "Messi", country: "Argentina", number: 10 },
  { name: "Mbappé", country: "France", number: 10 },
  { name: "Endo", country: "Japan", number: 6 }
];
```
